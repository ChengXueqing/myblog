## 常见问题速查

### ❌ 推送代码时认证失败
**原因**：GitHub 不支持密码认证  
**解决**：使用 [Personal Access Token](https://github.com/settings/tokens)  
**参考**：`GITHUB_DEPLOYMENT_GUIDE.md` → 常见问题 Q1

### ❌ GitHub Pages 显示 404
**原因**：
1. Actions 还在运行
2. 仓库是 Private（需要 Public）
3. `baseURL` 配置错误

**解决**：
1. 等待 1-3 分钟
2. 在仓库 Settings → Danger Zone 改为 Public
3. 检查 `hugo.toml` 中的 `baseURL`

**参考**：`GITHUB_DEPLOYMENT_GUIDE.md` → 常见问题 Q2

### ❌ 博客样式丢失
**原因**：`baseURL` 不匹配  
**解决**：确保以下两处配置一致：
1. `hugo.toml`: `baseURL = "https://chengxueqing.github.io/myblog/"`
2. `hugo.yml`: `hugo --minify --baseURL "https://chengxueqing.github.io/myblog/"`

**参考**：`GITHUB_DEPLOYMENT_GUIDE.md` → 常见问题 Q3

## 双远程仓库配置（可选）

如果你想同时保留 Gitee 和 GitHub：

```bash

git push origin master

# 推送到 GitHub
git push github master
```

**建议**：
- 主要使用 GitHub（已配置自动部署）
- Gitee 作为备份（可选）
