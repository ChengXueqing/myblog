# 基础认知

> 知道"怎么用"只是开始，知道"为什么这么设计"才算入门。

## 学什么

- **JS 语言设计** — 为什么有闭包、原型链、事件循环，这些问题是怎么来的
- **CSS 布局本质** — 为什么 Flex/Grid 这样设计，它们在解决什么排版问题
- **HTTP 协议哲学** — 无状态、缓存、幂等，这些设计背后的取舍
- **Git 的版本控制哲学** — 快照 vs 增量，分支的本质是什么

## 不写什么

- ❌ API 用法——查 MDN 更快
- ❌ 配置步骤——记在代码里
- ❌ 速查表——AI 随问随答
